# SkillSnap 🎮
### Gamified Career Discovery Platform

Play skill-based mini-games across 17 career paths and discover 
which career matches your strengths.

**Built with:** Java 17 · JavaFX 21 · MySQL 8 · Maven

**Developer:** Muhammad Usman Saleem | CMSID: 540398 | NUST SEECS

## Features
- 51 mini-games across Tech, Finance, Healthcare, Engineering
- Smart career matching based on game performance
- XP system, badges, daily quests, leaderboard
- Friends & social with XP comparison
- Career roadmaps with senior industry advice

## How to Run (Developers)
1. Clone the repo
2. Set up MySQL and run `schema.sql`
3. Add `mysql-connector-j-8.0.33.jar` to project libraries
4. Add JavaFX VM args: `--add-modules javafx.controls,javafx.fxml,javafx.geometry`
5. Run `Main.java`
